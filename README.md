# G-LoG bi-filtration for medical image classification

Python scripts for the computation of G-LoG bi-parameter persistence filtration on Medmnist dataset ([Zenodo](https://doi.org/10.5281/zenodo.10519652)). Our bi-filtrations are built by cubical complexes implemented through the [Multipers](https://davidlapous.github.io/multipers/) and the [GUDHI](https://gudhi.inria.fr/) libraries.

We provide Jupyter notebooks to generate bi-parameter persistence modules for 2D and 3D medical images. Additionally, we include an example using the AdrenalMNIST3D dataset for generating [multi-parameter persistence images](https://proceedings.neurips.cc/paper/2020/hash/fdff71fcab656abfbefaabecab1a7f6d-Abstract.html) and performing classification using a Multi-Layer Perceptron (MLP).
- ```generate_convolve_log1_Medmnist2d_28_pickle.ipynb``` Generate muti-parameter persistence modules from Medmnist2D datasets and save their corresponding pickles. The parameters for the Guassian kernel are set to $\sigma=0, 0.5, 1, 1.5$, while the $\sigma=1$ for Laplacian of Gaussian.
- ```generate_convolve_log1_Medmnist3d_28_pickle.ipynb``` Generate muti-parameter persistence modules from Medmnist3D datasets and save their corresponding pickles. The parameters for the Guassian kernel are set to $\sigma=0, 0.5, 1, 1.5$, while the $\sigma=1$ for Laplacian of Gaussian.
- ```get_adrenal_pers_s05.ipynb``` provides an example of generating multi-parameter persistence images from multi-parameter persistence modules with Gaussian kernel $\sigma=0.5$ on Adrenalmnist3d dataset. The generated images are saved in ```pers_s05_log1.zip```.
- The notebook ```train_pers_adrenal_s05_log1_001_save_selfauc.ipynb```, located in the ```train_pers_s05_log1_001_selfauc```directory, provides an example of training an MLP classifier on multiparameter persistence images.

For convenience, we provide pickled multiparameter persistence modules (G-LoG with Gaussian kernel $\sigma=0.5$, AdrenalMNIST3D) to support other vectorizations like [multiparameter persistence landscape](http://jmlr.org/papers/v21/19-054.html) and [signed baecodes](https://proceedings.neurips.cc/paper_files/paper/2023/hash/d75c474bc01735929a1fab5d0de3b189-Abstract-Conference.html). The persistence modules can be downloaded here [Adrenalmnist3D_pickle](https://drive.google.com/file/d/1u5evmRaFilsYQdMGSRHJwOOyDlb80sLK/view?usp=drive_link).
